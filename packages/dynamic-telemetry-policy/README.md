# @opentelemetry/dynamic-telemetry-policy

OpenTelemetry Dynamic Telemetry Policy for Node.js — enables runtime updates to trace sampling rates, export toggles, and polling intervals via file or OpAMP sources.

## Installation

```sh
npm install @opentelemetry/dynamic-telemetry-policy
```

## Usage

See the [API documentation](./src/index.ts) and examples in this repository.
